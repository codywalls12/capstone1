Graph Sonification

The goal for this project was to allow the user to upload create a graph using their preferred method and have the ability to have the data be presented to them using sonification. The project so far has the ability for the user to create the graph using a file upload or enter the data manually. The project is being developed by a team of 5 during the Spring 2024 semester.
